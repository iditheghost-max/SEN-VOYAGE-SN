# README.md

## Plateforme d’analyse de données et de comptabilité

Ce projet vise à fournir une solution d’analyse de données et de gestion comptable pour les entreprises de commerce.

### Structure du projet
- `importation_donnees/` : Scripts pour importer les données (ventes, achats, etc.)
- `analyse/` : Scripts d’analyse des données
- `rapports/` : Génération de rapports
- `gestion_achats_ventes/` : Gestion des opérations d’achats et de ventes
- `main.py` : Point d’entrée de l’application

### Dépendances principales
- pandas
- numpy
- matplotlib

### Lancement
```bash
python main.py
```
